# BTTP Marketing Videos

Remotion compositions for polished product/marketing videos.

## Farmers Market detour

This composition visualizes the Route 11 Farmers Market detour described in `docs/AUTO-DETOUR-DETECTION.md`:

- closure near Mulcaster Street between Collier Street and Worsley Street
- likely path using Owen Street and McDonald Street
- rider-facing alert card and detour explanation

## Commands

```bash
cd marketing-videos
npm install
npm run studio
npm run still:farmers
npm run render:farmers
```

Outputs are written to `marketing-videos/out/`.
