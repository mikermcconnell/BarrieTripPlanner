# Archive Guide

This folder contains non-default historical context that has been removed from the repo root.

## What Belongs Here

- early project and implementation plans
- one-shot or greenfield prompts
- superseded feature/backlog strategy docs
- other files that are still useful for history but should not compete with current repo entrypoints

## How To Use This Folder

- Start with `AGENTS.md` and `README.md` before reading anything here.
- Treat files here as background, not as the current contract.
- If an archived doc conflicts with current code or durable docs, prefer current code and durable docs.
